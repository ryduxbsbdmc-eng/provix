Title: Provix — File Manager
Short description: Dual-pane file manager with git, archives, AI, and terminal.
Full description: |
  Provix is a powerful file manager for Android with dual-pane browsing, tabs,
  bookmarks, search, archive support (zip/rar/7z), git integration, folder compare,
  optional AI automation, and a built-in shell with Termux bridge.

  Features:
  - Copy, move, delete, rename, new folder
  - Content search (grep) and filename search
  - File preview for text and images
  - Themes and localization (English, Russian, Ukrainian)
  - Open/share files via system intents
