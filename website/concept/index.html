Возможности

Всё, что нужно в проводнике

Provix не заменяет ваш workflow — он встраивается в него. Двойные панели, быстрая навигация и инструменты разработчика без переключения окон.



Двухпанельный режим

Копируйте и сравнивайте папки бок о бок. Разделители с подсветкой при наведении — как в приложении.



Git прямо в списке

Цветовые метки modified, added, deleted. Статус репозитория виден без отдельного клиента.



PowerShell-терминал

Встроенная консоль по Ctrl+`. Работает в текущей папке — не нужен отдельный терминал.



Архивы и шифрование

ZIP, 7z, RAR и другие форматы через SharpCompress. Шифрование архивов.



ИИ-ассистент

Работа с файлами и контекстом проекта без выхода из проводника.
