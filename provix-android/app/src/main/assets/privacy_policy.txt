Provix Privacy Policy

Provix is an offline-first file manager. We do not collect personal data on device.

Optional features:
- AI automation sends prompts to providers you configure (OpenRouter, Ollama, LM Studio). No data is sent unless you enter API credentials and run AI commands.
- Internet permission is used only for optional AI and opening external links.

Storage access is used solely to browse and manage files you select.

Contact: support via GitHub repository issues.
